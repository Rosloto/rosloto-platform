# Rosloto Platform

Official repository for the Rosloto iGaming platform.

## Features

- Online casino platform
- API integrations
- Payment systems
- Admin dashboard
- User authentication
- Multi-language support

## Tech Stack

- Frontend: React / Vite
- Backend: Node.js
- Database: PostgreSQL
- Docker support
- CI/CD with GitHub Actions

```bash
git clone https://github.com/Rosloto/rosloto-platform.git
